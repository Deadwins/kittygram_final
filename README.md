# KittyGram

## Описание проекта

KittyGram - это веб-приложение для обмена фотографиями котиков. Пользователи могут загружать фотографии, указить имя, дату рождения, цвет и даже достижения своего котика. Это приложение создано для любителей котиков, чтобы делиться своими милыми фотографиями.

## Использованные технологии

Проект KittyGram был разработан с использованием следующих технологий:

- **Django**: Django был использован для создания серверной части приложения.
- **React**: React использовался для разработки клиентской части приложения.
- **Docker и Docker Compose**: Для контейнеризации приложения и управления его компонентами.
- **PostgreSQL**: Как база данных для хранения информации о пользователях, фотографиях и комментариях.
- **Nginx**: Для проксирования запросов к серверу и обслуживания статических файлов.
- **GitHub Actions**: Для автоматизации процессов сборки и развертывания.
- **HTTP-сервер http-server**: Для предоставления статических файлов клиентской части.
